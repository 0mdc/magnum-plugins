This repository contains plugins for Magnum OpenGL 3 engine.

INSTALLATION
============

Dependencies
------------

 * CMake    - for building
 * Magnum engine (https://github.com/mosra/magnum)
 * Qt       - some plugins need it

Compilation, installation
-------------------------

    mkdir -p build
    cd build
    cmake -DCMAKE_INSTALL_PREFIX=/usr .. && make
    make install
